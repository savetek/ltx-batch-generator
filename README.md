# 🎬 LTX-2 일괄 영상 생성기

Replicate API를 사용해 LTX-2로 영상을 일괄 생성하는 도구입니다.

## 기능

- 📝 텍스트 → 영상 (여러 프롬프트 한번에)
- 🖼️ 이미지 → 영상 (여러 이미지 한번에)
- 📦 ZIP 일괄 다운로드

## 사용법

1. [Replicate](https://replicate.com)에서 API 키 발급
2. API 키 입력
3. 프롬프트 또는 이미지 업로드
4. 생성 시작!

## 비용

- LTX-2 Distilled: $0.02/초
- 4초 영상 1개: 약 $0.08
